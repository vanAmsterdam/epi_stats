Note that the estimate is equal for spearman vs pearson of rank, however, the p-value for pearson on the ranks is lower.

Is this always the case?

set.seed(2)
nsim = 100
n = 20
noise = .4

sim <- data.frame(
  rp  = numeric(nsim),
  rs  = numeric(nsim),
  rpr = numeric(nsim),
  blm = numeric(nsim),
  sqr_rsq = numeric(nsim),
  pp  = numeric(nsim),
  ps  = numeric(nsim),
  ppr = numeric(nsim),
  plm = numeric(nsim)
)

for (i in 1:nsim) {
  x = runif(min = -1, max = 1, n = n)
  y = x + rnorm(n, sd = noise)
  cor_p  <- cor.test(~x+y, method = "p")
  cor_s  <- cor.test(~x+y, method = "s")
  cor_pr <- cor.test(~rank(x)+rank(y), method = "p")
  lm_fit <- lm(y~x)
  sum_fit<- summary(lm_fit)
  sim[i, ] <- c(cor_p$estimate, cor_s$estimate, cor_pr$estimate, 
                lm_fit$coefficients[2],
                sqrt(sum_fit$r.squared),
                cor_p$p.value, cor_s$p.value, cor_pr$p.value,
                sum_fit$coefficients[2,4])
}

plot(rs~rpr, data = sim)

plot(ps~ppr, data = sim)

plot(rp~sqr_rsq, data = sim)

plot(sim$ps-sim$ppr)

plot(sim$pp-sim$plm)

Conclusions

  • Pearson correlation = \(R^2\) from simple linear regression with intercept, and p-values are equal
  • Spearman correlation = Pearson on the ranks of the data, p-values differ in that the spearman p-values are sometimes higher.
